Ansible服务简介

软件由python语言开发，其功能实现基于SSH远程连接服务，可以实现批量系统配置、批量软件部署、批量文件拷贝、批量运行命令等功能

ansible软件特点

   a 不需要单独安装客户端（no agents），基于系统自带的sshd服务，sshd就相当于ansible的客户端

   b 不需要服务端(no servers)

   c 需要依靠大量的模块实现批量管理。

   d 配置文件/etc/ansible/ansible.cfg,不用配置

ansible颜色区别

ansible颜色信息显示说明：

翔黄色：对远程节点进行相应修改

帽子绿：对远程节点不进行相应修改，或者只是对远程节点信息进行查看

深红色：操作执行命令有异常

浅紫色：表示对命令执行发出警告信息（可能存在的问题，给你一下建议）

ansible命令执行过程

  1. 加载自己的配置文件 默认/etc/ansible/ansible.cfg
  2. 加载自己对应的模块文件，如：command
  3. 通过ansible将模块或命令生成对应的临时py文件，并将对应文件传输到远程服务器的对应执行用户$HOME/.ansible/tmp/absible-tmp-数字/*.py文件
  4. 给文件+x执行权限
  5. 执行并返回结果
  6. 删除临时py文件，sleep 0退出
