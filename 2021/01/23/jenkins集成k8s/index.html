jenkins集成k8s集群

  • 创建证书

打开~/.kube/config文件

  • jenkins创建证书凭证

  • 集成k8s

选择系统管理 => 系统配置 => Cloud => 点击a separate configuration page.
点击连接测试,如出现success(老版本)或者Connected to Kubernetes v1.X.X,表示连接k8s集群成功
